About

This app is my bac diploma project. It is made with MEAN Stack (MongoDB, Express.js, Angular.js and Node.js) technology. I use Gulp.js to compile SASS files, BrowserSync for frontend and Git for version controle. The app also uses ServiceWorker to cache some files for offline use and native JavaScript Notification API to notify about prepared order.

Using the app the user can order food in the restaurants, book for a specific time and don't wait in the queue. After booking they get a code to show the seller.
